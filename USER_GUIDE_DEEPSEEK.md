## Support & Feedback

### Common Questions

**Q: Can I regenerate AI content if I don't like it?**  
A: Yes! Click the generate button again for a new variation.

**Q: Does language switching affect existing content?**  
A: No, but you can regenerate any phase content in the new language.

**Q: Is my data private?**  
A: Yes, all data is stored locally in your browser using Spark's KV storage.

**Q: How accurate is the AI validation?**  
A: The AI checks against industry standards (HIPAA, WCAG 2.1 AA) and provides 85-95% accuracy in identifying issues.

**Q: Can I edit AI-generated content?**  
A: Absolutely! All AI content is editable. Use AI as a starting point, then customize.
